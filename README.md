## ✨ Özellikler

### 📚 13 Ders İçeriği
- **TYT**: Türkçe, Fizik, Kimya, Biyoloji, Tarih, Coğrafya, Felsefe
- **AYT**: Fizik, Kimya, Biyoloji, Tarih, Coğrafya, Felsefe Grubu

### 🔮 Tahmin Soruları
- 80+ tahmin sorusu geçmiş yıl analizi bazlı
- Her sorunun çıkma olasılığı (%85-98)
- 🔴 HER YIL / 🟡 SIKÇA / 🟢 ARADA / ⚠️ PİF etiketleri
- Pif noktaları ve tuzak soru uyarıları

### 🎯 Dinamik Net Projeksiyonu
- TYT ve AYT ders bazlı tahmini net hesaplama
- Mevcut performansa göre sınav simülasyonu
- Motivasyon mesajları ve günlük ipuçları

### 👨‍🏫 Öğretmen Paneli
- Öğrenci bazlı ilerleme takibi
- Başarı sıralaması (🥇🥈🥉)
- Ders bazlı detaylı rapor ve net hesaplama
- Son aktiflik takibi

### 🔐 Kullanıcı Sistemi
- Öğrenci kayıt/giriş sistemi
- Kullanıcı bazlı ilerleme kaydetme
- Öğretmen hesabı ile tüm öğrencileri görüntüleme

### 🎨 Premium Tasarım
- Dark glassmorphism tema
- Responsive (mobil + tablet + PC)
- SVG animasyonlu progress ring
- Mikro-animasyonlar ve hover efektleri
- Premium scrollbar ve glass polish

## 🚀 Kurulum

```bash
# Repo'yu klonla
git clone https://github.com/KULLANICI_ADINIZ/yks-2026-calisma-asistani.git

# Klasöre gir
cd yks-2026-calisma-asistani

# Tarayıcıda aç (herhangi bir sunucu ile)
npx -y http-server -p 8080 -o

# Veya doğrudan index.html'i tarayıcıda aç
```
